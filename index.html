Javascript

JavaScript adalah bahasa pemrograman tingkat tinggi yang umumnya digunakan untuk mengembangkan aplikasi web interaktif. Ia adalah salah satu komponen utama dari teknologi web, bersama dengan HTML (Hypertext Markup Language) dan CSS (Cascading Style Sheets).

Saat ini Javascript tidak hanya bisa di jadikan sebagai pemanis antarmuka saja tapi juga dapat mengolah data dari database dan mengatur server web.

Dokumentasi

Figma

Figma adalah alat desain berbasis web yang digunakan untuk membuat desain antarmuka pengguna (UI) dan desain pengalaman pengguna (UX). Figma memungkinkan tim desain dan pengembangan untuk berkolaborasi secara real-time dalam lingkungan yang terintegrasi.

Figma telah menjadi alat yang populer di kalangan desainer dan pengembang, terutama karena kemampuannya untuk membuat proyek desain antarmuka pengguna yang kompleks dan desain dengan efisiensi tinggi.
